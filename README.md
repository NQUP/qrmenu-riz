# QR Menu Riz

Aplikasi QR Menu berbasis Laravel untuk manajemen menu dan pemesanan.

## Stack

- Laravel
- MySQL
- Vite

## Menjalankan Project

```bash
cp .env.example .env
composer install
php artisan key:generate
php artisan migrate
npm install
npm run dev
php artisan serve
```
